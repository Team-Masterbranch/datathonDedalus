Para genarar las consultas te enviamos una esquema de la base de datos con:
- Nombre de la columna
- Tipo de dato
- Información estadística cuando sea aplicable:
  * Para campos numéricos: valores mínimos, máximos y promedio
  * Para campos categóricos: número de valores únicos, lista de valores posibles
  * Conteo de valores faltantes
  * Características de distribución

Utiliza estos metadatos para:
1. Elegir los campos apropiados para las consultas
2. Validar rangos de valores en los filtros
3. Seleccionar visualizaciones significativas
4. Sugerir agregaciones relevantes
